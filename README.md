plotly

## example

## Prerequisites

1. Install [Trunk](https://trunkrs.dev/) using `cargo install --locked trunk`.

## How to Run

1. Run `trunk serve --open` in this directory to build and serve the application, opening the default web browser automatically.

```sh
trunk serve --open
```
